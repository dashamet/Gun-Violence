    We used three data sources to create the visualizations featured on this website:
    1. CDC WISQARS Data - We used this data for the line graph breaking down causes of death and injuries over time.
    2. Gun Violence Archive - We obtained two datasets from this source: youth injured by firearms and youth killed by firearms, both in 2019. We used these datasets to create the animated dot visualization and the map with points representing victims.
    3. RAND State Firearm Law Database - We relied on this extensive dataset of states' firearm policies for our choropleth map showing the correlation between policies and gun deaths.
